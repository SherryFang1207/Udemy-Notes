## Selectors
There are three kinds of selectors: Tag, Class, ID.   
#### Syntax of Tag Selector:
```
Tag {
  Property: Value;
}
```
#### Syntax of Class Selector: (adding `class` attribute)
```
.class {
  Property: Value;
}
```
#### Syntax of id Selector: (adding `id` attribute)
```
#id {
  Property: Value;
}
```
